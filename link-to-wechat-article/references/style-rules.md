# Style Rules

Apply these defaults unless the user overrides them.

## Voice

- Write like a person with a point of view.
- Keep it calm, direct, and opinionated.
- Commentary first, summary second.
- Avoid any obvious AI-summary cadence.

## Forbidden backstage wording

Do not write phrases like:
- 原文
- 链接
- 这篇文章里提到
- 上面这篇
- 我看了一个链接
- 以下是要点

The published article must read like a standalone finished piece, not a note about the writing process.

## Paragraph rhythm

- Do not break lines after commas.
- Prefer paragraph breaks after full stops or completed thoughts.
- Keep paragraphs short for mobile reading.
- Avoid bullet lists unless the user explicitly asks.

## Structure

Good default flow:
1. Open with a judgment, tension, or reaction.
2. Explain what happened.
3. Explain why it matters.
4. Add a sharper personal view.
5. End with a light closing sentence.

## Length

Default to medium length: roughly 900–1500 Chinese characters.

## Ending

Use a light closing sentence.
Do not end with preachy CTA copy unless the user explicitly asks.
