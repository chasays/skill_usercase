# skill_usercase

A public collection of reusable OpenClaw / Agent skills.

## Included skills

- `link-to-wechat-article` — turn a shared link into a WeChat Official Account draft article, rewritten in a personal-commentary style.

## Repository structure

```text
skill_usercase/
  link-to-wechat-article/
    SKILL.md
    LICENSE
    references/
    scripts/
```

## Notes

- Each skill lives in its own subdirectory.
- More skills can be added over time under the same repo.
- Skill-specific usage and requirements are documented in each skill's `SKILL.md`.
