# skill_usercase

A public collection of reusable OpenClaw / Agent skills.

一个公开的 OpenClaw / Agent Skills 合集仓库。

## Included skills / 已收录技能

| Skill | Description |
|---|---|
| `link-to-wechat-article` | Turn a shared link into a WeChat Official Account draft article, rewritten in a personal-commentary style. / 将分享链接改写成带个人评论风格的微信公众号草稿文章。 |

## Repository structure / 仓库结构

```text
skill_usercase/
  README.md
  link-to-wechat-article/
    SKILL.md
    LICENSE
    references/
    scripts/
```

## Conventions / 约定

- Each skill lives in its own subdirectory.  
  每个 skill 单独放在自己的子目录里。
- Each skill should include its own `SKILL.md`.  
  每个 skill 应自带自己的 `SKILL.md`。
- If needed, each skill can also include `references/`, `scripts/`, `assets/`, and examples.  
  如有需要，每个 skill 也可以包含 `references/`、`scripts/`、`assets/` 和示例文件。
- More skills can be added over time under this same repository.  
  后续可以继续在这个仓库下增加更多 skill。

## How to use / 使用方式

Read the `SKILL.md` inside each skill directory for usage, requirements, and workflow details.  
具体用法、依赖和工作流说明，请直接查看对应 skill 目录下的 `SKILL.md`。
